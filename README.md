# Internship Search Platform Assignment 🚀

A modern internship search platform inspired by Internshala, built using React.js, Vite, Tailwind CSS CDN, and REST APIs.

# 📌 Features

✅ Internship listing using API
✅ Frontend-only filtering
✅ Search internships
✅ Responsive UI/UX
✅ Sticky filter sidebar
✅ Professional internship cards
✅ Loading state
✅ Empty state UI
✅ Modern navbar
✅ Mobile responsive design

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/internship-search-platform-assignment.git
```

---

## 2️⃣ Navigate to Project

```bash
cd internship-search-platform-assignment
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Run Development Server

```bash
npm run dev
```

# 🎯 Filtering Features

Users can filter internships based on:

* Profile
* Location
* Duration
* Stipend

All filtering is handled completely on the frontend without additional API requests.
